# 插件改名并集中默认配置

## 目标

- 把插件名称改成 `IkunCode Pulse`。
- 给这几个设置项补上默认值：
  - `ikuncode-pulse.baseApiUrl = https://api.ikuncode.cc/api/`
  - `ikuncode-pulse.siteName = IkunCode`
  - `ikuncode-pulse.siteUrl = https://api.ikuncode.cc/`
- 同时保留以后切换其他 token 商的灵活性。

## 实施方式

- 新增一个集中配置文件，专门存“品牌名”和“默认站点信息”。
- 运行构建时自动把这些默认值同步到 `package.json`，避免以后手改很多地方。
- 扩展运行时代码也从同一个集中配置读取默认值和显示名称。

## 验证

- 编译通过。
- 打包后的 VSIX 名称、扩展显示名、设置默认值都正确。
