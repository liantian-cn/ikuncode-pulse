# 清理旧命名并统一为 ikuncode-pulse

## 目标

- 排查当前插件不能用的原因。
- 删除所有旧命名残留。
- 统一项目、设置键、命令 ID、文档示例为 `ikuncode-pulse`。

## 已发现问题

- 插件显示名已经改成 `IkunCode Pulse`，但内部很多地方还在使用旧命名。
- 这些旧名字同时出现在设置项、命令 ID、代码配置读取、tooltip 命令链接里。
- 这样会导致“显示名、设置项、代码逻辑”三边不一致，容易直接失效或让用户找不到设置。

## 修改方向

- 配置分组统一改成 `ikuncode-pulse`。
- 命令 ID 统一改成 `ikuncode-pulse.xxx`。
- 文档示例和同步脚本同步改名。
- 完成后重新编译和打包，确认没有旧命名残留。
